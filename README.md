# 🚀 Симулятор космического корабля (C++)

Мой самый первый учебный проект на C++. Игра создана для закрепления базовых конструкций языка: циклов `while`, ветвлений `if-else`, оператора `switch-case` и генерации случайных чисел. 

В процессе разработки были успешно найдены и исправлены логические баги с «эффектом памяти» переменных и порядком валидации условий финала.

---

## 🎮 Как играть?
Вы управляете грузовым кораблем, летящим на Марс. Ваша цель — преодолеть **1000 км**, сохранив запасы топлива и прочность обшивки.

Каждый ход вам доступен выбор из 3 действий:
* `1` — **Полная скорость**: мощный рывок вперед (-250 км), но расходует много топлива (-30 литров).
* `2` — **Экономный режим**: продвижение на 100 км с минимальными затратами ресурсов (-10 литров).
* `3` — **Ремонт корабля**: остановка на один день для починки щита (+25% прочности) за счет небольшого расхода топлива (-5 литров).

---

## 🎲 Ночные события
После каждого выбора капитана наступает космическая ночь. Генератор случайных чисел запускает одно из трех событий:
1. ✨ **Спокойная ночь** — ресурсы не изменяются.
2. ☄️ **Атака астероидов** — щит корабля получает случайный урон от 1 до 99 единиц.
3. 🌀 **Пространственная аномалия** — корабль переносится вперед на случайное расстояние до 50 км.

---

## ⚙️ Условия завершения
* 🏆 **Победа**: Расстояние сократилось до 0 км (или меньше) — корабль успешно сел на Марсе.
* 💀 **Поражение**: Топливо или прочность щита опустились до 0.
* 🛑 **Критический сбой**: Ввод недопустимой команды в меню уничтожает корабль из-за нарушения инструкций.
